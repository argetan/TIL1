## 목록

- 
- 하이픈을 입려가고 스페이스바 누르면 순서없는 목록이 만들어진다.
- 그리고 엔터를 누르면 하나 더 생성
  - 엔터 누르고 탭을 누르면 하위 목록을 생성가능함
  - 엔터 누르고 계속 쓰다가 나오고 싶다면
- 쉬프트 탭 눌르면 나올 수 있음
- 만약 완전히 끝내고 싶다면 엔터를 두번

1. 숫자 누르고 .마침표 찍은 다음 스페이스 바 누르면 만들어짐
2. 엔터 누르면 다음 목록
   1. 탭 키를 눌러 하위목록 생성
   2. 나가고 싶으면 쉬프트 탭 눌러서 밖으로 나올 수 있음

내보내기 가서 pdf로도 생성가능해짐
